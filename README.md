# RandomUser

## Descrizione

**RandomUser** è un sito web che permette di generare e visualizzare un numero variabile di utenti casuali utilizzando la **Random User API**.

Gli utenti vengono recuperati tramite chiamate all’API e mostrati a schermo sotto forma di **card**, ognuna contenente le principali informazioni personali.

## Funzionalità principali

- **Generazione di utenti casuali**  
  È possibile scegliere il numero di utenti da generare, ottenendo risultati sempre diversi a ogni richiesta.

- **Integrazione con Random User API**  
  I dati vengono caricati dinamicamente e aggiornati in tempo reale in base ai parametri scelti.

- **Visualizzazione tramite card**  
  Ogni utente è rappresentato da una card che organizza le informazioni in modo chiaro e leggibile.
  
- **Salvataggio e Visualizzazione degli utenti preferiti**  
  E' possibile salvarsi gli utenti preferiti, che verranno mostrati in un apposita lista a lato

- **Filtri di ricerca**  
  Gli utenti possono essere richiesti applicando filtri come genere, nazionalità o altri parametri supportati dall’API, permettendo una generazione più mirata.
